# Coptic to English Translator!

Hey yall! This is a Coptic to Coptic-English translator, meaning if you paste something in Coptic, it will translate for you into Coptic-English. Use this for your hymns class or yourself; just paste in the coptic for one of the hymns, copy the coptic-english into another doc and sing it out loud with Youtube or your instructor!

## Notes

- This app is phonetic, not a meaning translation.
- It runs entirely in the browser, so you do not need to install anything.
